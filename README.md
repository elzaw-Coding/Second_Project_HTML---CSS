# Kasper Agency - Creative Portfolio (Project #02)

**Kasper** is a modern, one-page creative portfolio/agency template. This project is the second milestone in my web development journey, focusing on advanced CSS layouts, precision in positioning, and building a fully responsive design from scratch.

---

## 🚀 [Live Demo](https://elzaw-coding.github.io/Second_Project_HTML---CSS/) 

## 🎯 Project Goals & Learning Outcomes
Building on the foundation of the Leon template, Kasper introduced more complex UI challenges:
- **Advanced Positioning:** Mastered the use of `absolute` and `relative` positioning for layered elements (like the landing text and section markers).
- **Reusable Components:** Created a global `.main-heading` system using `::before` and `::after` pseudo-elements for consistent design across sections.
- **Overlay & Glassmorphism:** Implemented semi-transparent overlays (`rgba`) and handled `z-index` layers to ensure content visibility over high-quality background images.
- **Responsive Architecture:** Built a fluid layout that adapts to Mobile, Tablet, and Desktop using custom Media Queries (without external frameworks).
- **Interactive Elements:** Added smooth hover effects, transitions, and styled form elements (Inputs/Buttons).

## 🛠️ Built With
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** - **Custom Variables:** Used for a unified color palette and easy maintenance.
  - **Flexbox & Grid:** Combined for structural layout and internal component alignment.
  - **Pseudo-Elements:** For decorative shapes and iconography.
- **Font Awesome:** For professional vector icons.
- **Google Fonts:** Utilizing "Open Sans" for a clean, modern look.
